enum lk_table_shape

Flags controlling the output keymap.

Enumerator:
LK_SHAPE_DEFAULT 
LK_SHAPE_FULL_TABLE 

one line for each keycode

LK_SHAPE_SEPARATE_LINES 

one line for each (modifier,keycode) pair

LK_SHAPE_UNTIL_HOLE 

one line for each keycode until 1st hole
